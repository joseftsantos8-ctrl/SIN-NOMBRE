# PROJECT INFO

**Project**: Sirena Market Vista Cana SP13 (SIN-NOMBRE)
**Stack**: HTML5, CSS3 (Vanilla), JavaScript (Vanilla)
**Architecture**: Single Page Application (SPA) with DOM manipulation for view routing and role-based access control.

## Overview
A web application for the operational management of the Sirena Market Vista Cana SP13 store. It includes role-based access control, task assignment, inventory management, and user management.
